smallworld.js

A small utility for generating small world maps.

Why?

  • No heavy client libraries
  • No tile downloads
  • No license restrictions
  • Speedy canvas rendering
  • B.Y.O.G. (Bring Your Own GeoJSON)

Default

Just pass in your GeoJSON source and you're done.

$('.js-basic').smallworld({
	geojson: data
});

Custom Colors

Maps are rendered using native canvas elements, so setting custom colors for everything is easy.

$('.js-colors').smallworld({
	geojson: data,
	waterColor: '#acbcc9',
	landColor: '#f2e5d4'
});
